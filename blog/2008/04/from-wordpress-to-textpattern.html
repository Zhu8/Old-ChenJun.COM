从wordpress到textpattern

发布于： 2008年4月16日 14时29分04秒 Chen Jun(Zhu8)

其实很久以前就试用了 TXP ，但是一直是一头雾水，今天终于 其实昨天 就已经弄好了。不过一直wordpress导入有点小问题，所以一直没有换过来。现在折腾好了，做一下小 记录：

解决乱码问题 打开 import_wp.php ，找到下面这行： php mysql_select_db($b2db, $b2link); 在后面加上： php mysql_query(&#8220;SET NAMES &#8216;utf8&#8242;&#8221;, $b2link); * 增加 Excerpt 部分 找到下面这行： php p.post_content as Body, 在后面加上： php p.post_excerpt as Excerpt, 同样的，找到这行： php Body_html = &#8216;&#8221;.doSlash($Body_html).&#8221;&#8216;, 在后面加上： php Excerpt = &#8216;&#8221;.doSlash($Excerpt).&#8221;&#8216;, Excerpt_html = &#8216;&#8221;.doSlash($Excerpt).&#8221;&#8216;, * 不导入附件生成的页面 找到下面这行： php &#8220;.$wpdbprefix.&#8221;users as u on u.ID = p.post_author 在后面加上： php WHERE p.post_status != &#8216;attachment&#8217; 差不多就这些了，在懒懒喵的 旧文 中提到的留言关闭问题已经在Textpattern · 4.0.6中得到了解决。 但是我本人的一个问题，Tag问题还没有搞定，导入会变成Category，我是用的笨办法：直接在WP数据库中先删除它，哈哈！ 还有一个小问题，导入user错误，不过如果是单用户的Blog，大可不必去管这个问题。

*放出修改后的 import_wp.php

P.S 初次使用 Textile ，感觉还要适应好段时间，不过用起来确实是种不同的体验。
